Pathfinding algorithms

Dijkstra's

Dijkstra's algorithm (or Dijkstra's Shortest Path First algorithm, SPF algorithm)[2] is an algorithm for finding the shortest paths between nodes in a graph, which may represent, for example, road networks.

A*

A* (pronounced "A-star") is a graph traversal and path search algorithm, which is often used in many fields of computer science due to its completeness, optimality, and optimal efficiency.

BFS

Breadth-First Search (BFS) is a Graph Traversal Algorithm for traversing or searching tree or graph data structures. BFS solves Single Source Shortest Path problem.
